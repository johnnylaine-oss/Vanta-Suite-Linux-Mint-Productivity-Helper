# Vanta Suite — Keyboard Shortcuts

## Default Global Shortcuts

| Shortcut | Action |
|---|---|
| `Ctrl+Space` | Open Command Palette |
| `Ctrl+Shift+Space` | Toggle Quake Terminal |
| `Ctrl+Shift+S` | Open Settings |
| `Ctrl+Shift+C` | Open Clipboard |
| `Ctrl+Shift+N` | Open Notifications |
| `Ctrl+Shift+A` | Open AI Assistant |
| `Ctrl+Shift+F` | Toggle Focus Mode |
| `Ctrl+Shift+1` | Open Development Apps |
| `Ctrl+Shift+2` | Open Gaming Apps |
| `Ctrl+Shift+3` | Open Media Apps |

## Command Palette Navigation

| Key | Action |
|---|---|
| `Arrow Up/Down` | Navigate results |
| `Enter` | Execute focused command |
| `Escape` | Close palette |
| `Tab` | Cycle ambiguous matches |

## Rebinding Shortcuts

1. Open Settings (Ctrl+Shift+S)
2. Go to Shortcuts section
3. Click a binding and press the new key combination
4. Conflicts are shown and blocked — never silently overwritten

## Hotkey Profiles

Create named sets of bindings for different workflows (e.g., "Default", "Gaming", "Coding"):
1. Settings → Shortcuts → Profiles
2. Create a profile, customize bindings
3. Switch profiles to hot-swap all shortcuts without restarting

## Import/Export

Shortcut profiles can be exported as JSON files and imported on another machine.
