# Vanta Suite — User Guide

## Welcome

Vanta Suite is a keyboard-first productivity app for Linux Mint. It runs quietly in the background — press a keyboard shortcut and it appears, do what you need, and it gets out of your way.

## Command Palette

Press `Ctrl+Space` to open the Command Palette. Type any command, app name, or action:
- `timer 20m` — Start a countdown
- `chrome` — Launch Chrome
- `lock` — Lock the screen
- `settings` — Open settings
- `volume 50` — Set volume to 50%

Use arrow keys to navigate results, Enter to execute, Esc to close.

## Favorite App Groups

Organize your frequently used apps into named groups. Go to Settings → Favorites to create groups and add apps. Bind a hotkey to each group in Settings → Shortcuts.

## Terminal

A fast terminal wrapper around your normal shell (bash, fish, zsh). Supports tabs, split panes, and Quake-style dropdown mode.

## AI Assistant

A local AI assistant powered by Ollama. Nothing is sent to the cloud. It can organize files, create reminders, explain terminal errors, and more. Requires Ollama to be installed and running.

## Clipboard Manager

Every item you copy is saved. Press `Ctrl+Shift+C` or use the `clipboard` command to browse your clipboard history. Search, pin, and paste with the keyboard.

## Reminders

Create reminders in natural language ("tomorrow 2pm", "every Monday"). Snooze or mark as done. Recurring reminders supported.

## Notifications

All desktop notifications are captured and stored. Search, filter, and manage them from the Notifications panel. Enable Focus Mode to silence everything.

## Workflows

Automate multi-step tasks. Example: the built-in "Coding" workflow opens VS Code, a terminal, your browser, and switches to the right workspace — all with one shortcut.

## Settings

Every aspect of Vanta Suite is configurable from the searchable Settings screen. Export and import settings for backup or moving between machines.
